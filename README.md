## General info
Cocktails is a drink management web app. 

Create, edit, and delete simple drink recipies.

Upload beautiful images to accompany your recipe.
	
## Technologies
Cocktails was created with:
* Ruby on Rails
* JavaScript
* HTML/SCSS
* [Cloudinary](https://cloudinary.com/)
* Heroku
